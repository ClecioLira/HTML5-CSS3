Principais Formatações

Negrito/ Destaque

Nessa frase temos um termo em negrito usando a tag <B> <B>(não semantica).

Nessa frase tenho um termo em destaque usando a tag <STRONG> </STRONG>(semantica).


Italico /Ênfase

Nessa frase temos um termo em itálico usando a tag <I> </I>(não semantica).

Nessa frase temos um termo em ênfase usando a tag <EM> </EM>(semantica).


Texto Marcado

Podemos criar tambem um texto marcado usando a tag <MARK> </MARK>.

E no outro paragrafo temos outro texto marcado no final.


Texto grande e pequeno

Estamos criando um <BIG> texto grande </BIG> e um <SMALL> texto pequeno </SMALL>.


Texto deletado

Podemos criar <DEL> um texto como excluído </DEL> para indicar que ele deve ser lido, mas não considerado.


Texto inserido

Podemos marcar <INS> um texto como inserido </INS> para dar uma ênfase e indicar que ele foi adicionador depois.

Para inserir coisas do tipo <SUP> </SUP> x20+3
